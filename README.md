# Flask CRUD MySQL

A simple Flask application for performing CRUD (Create, Read, Update, Delete) operations on a MySQL database using the `Flask-SQLAlchemy` and `Flask-Marshmallow` extensions.

## Setup

### 1. Environment Setup:

```bash
mkdir flask_crud_mysql
cd flask_crud_mysql
pip install flask flask_sqlalchemy flask_marshmallow marshmallow-sqlalchemy mysqlclient
